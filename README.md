# Kana Voice Tool
## KanaVoice
KanaVoiceとは、Kanaサービスの音声合成です。  
独特ながら、あらゆる場面で使用可能な音源で大変便利です。  

## Kana Voice Tool
KanaVoiceを利用できるのがKana Voice Toolです!
